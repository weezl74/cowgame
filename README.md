
# Cowgame (React Native, JS)

Frontend-only React Native (Expo) puzzle (Block‑Jam mode) with an educational overlay:
answer climate/decarbonisation micro-quizzes to get power-ups (extra moves / hints).

## Scripts
- `npm start` — start Expo dev server
- `npm run android` — run on Android device/emulator
- `npm run ios` — run on iOS simulator (macOS)
- `npm run web` — run in browser (RN Web via Expo)

## Tech
- React Native (Expo)
- JavaScript + JSX (no TypeScript)
- Reanimated (simple fade animations)

## Files of interest
- `src/games/blockjam/*` — puzzle logic, levels, main screen
- `src/edu/*` — quiz modal and question data
